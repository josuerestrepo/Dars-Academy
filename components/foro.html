¿Para qué sirve el atributo "placeholder" en un input de HTML?

Respuesta: Usuario 1

Placeholder se usa para mostrar un texto dentro del campo de entrada que desaparece cuando el usuario empieza a escribir. Es útil para dar una pista de lo que se debe escribir.
